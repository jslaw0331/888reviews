# Performance baselines

Generated by `npm run build:all` → `scripts/record-baseline.mjs`.

Each JSON file records minified CSS/JS bundle sizes after a build. Compare across deploys to catch asset regressions.

## Manual Lighthouse checks (post-deploy)

| URL | Metrics to track |
|-----|------------------|
| `/` | TTFB, LCP, CLS, TBT |
| `/reviews` | TTFB, LCP |
| `/blackjack` | TTFB, transfer size |
| `/bonus/:slug` | TTFB (serverless detail route) |

Use [PageSpeed Insights](https://pagespeed.web.dev/) or Chrome DevTools Lighthouse.

## Expected improvements (Phase 1+)

- Hub page TTFB: static CDN vs serverless Express
- CSS transfer: split bundles (legal pages skip hub/detail)
- JS on legal pages: `static.*.js` (~0.6KB) vs full `core.*.js` (~264KB minified)
- Nav paint: inlined header/footer (no fetch waterfall)
